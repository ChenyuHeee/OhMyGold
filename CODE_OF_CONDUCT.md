# 行为准则

本项目遵循开放、包容、尊重的社区文化。所有贡献者和参与者需遵守以下原则：

1. **尊重他人**：禁止任何形式的歧视、骚扰或人身攻击。不同观点可以讨论，但请保持礼貌。
2. **积极协作**：欢迎对问题的建设性反馈和改进建议。提交 Issue/PR 时请清晰描述上下文、复现步骤与预期结果。
3. **遵守法律与许可**：提交的代码和文档应为原创或已获得授权，严禁上传侵权内容。
4. **关注安全**：如发现潜在安全问题，请参考 `SECURITY.md` 中的披露流程，与维护者直接沟通。

违反上述行为准则的参与者可能会被限制贡献资格，具体由维护团队裁量。感谢你的合作与支持！
