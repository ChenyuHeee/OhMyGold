# 安全披露指南

感谢你愿意帮助我们提升安全性。如果你发现潜在的安全漏洞，请按照以下流程私下披露，以便我们及时修复。

1. **私下报告**：请发送邮件至 `security@ohmygold.dev`，标题包含“Security Disclosure”。
2. **提供细节**：
   - 漏洞简要描述及影响范围；
   - 复现步骤或 PoC（如有）；
   - 受影响的版本或配置；
   - 你的联系方式（可选）。
3. **等待确认**：我们将在 72 小时内回复，并与您协作验证与修复。
4. **公开披露**：在修复发布前，请不要公开披露漏洞细节。修复完成后，我们会在发布说明中致谢（除非你要求匿名）。

感谢你为社区安全做出的努力！
